# Hi, I'm Zubash 👋

Fullstack software engineer from Pakistan 🇵🇰. I build things with **C# and .NET** on the backend and **React / React Native** on the front — the kind of software that's actually maintainable, not just functional.

I think a lot about architecture and system design. Good abstractions, clean patterns, knowing *why* a structure works and not just *that* it works. Data structures and algorithms are part of that mindset — they're how I reason through problems, not something I dust off for interviews.

Outside of work, I'm drawn to computer graphics — shaders, render pipelines, how light behaves in real-time environments, a bit of game dev. The intersection of math and visual output is endlessly interesting to me.
